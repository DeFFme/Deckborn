# Deckborn

Deckborn is a tactical turn-based card game inspired by lane-based combat systems.

The project is currently in active development.

Inspired by The Elder Scrolls: Legends
---

## Current features

- Turn-based card combat system
- Lane-based battlefield mechanics
- AI opponent logic
- Support card type
- Weapon card type
- Keywords system 
- Shop system prototype
- Local multiplayer prototype
- Glossary system

---

## Planned features

- Difficulty selection
- Additional unique card abilities
- Online multiplayer
- Campaign mode
- Expanded card collection
